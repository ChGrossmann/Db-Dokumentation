Aufgabe 3 - 30 Punkte

Erstellen Sie eine Java Web-Lösung, die folgenden Java Klassen besitzt:

  • dao.DBConnection
  • dao.ArtikelDao
  • beans.Artikel

Die Klasse ArtikelDao sollte folgende Methoden enthalten:

  • allArtikel()
  • boolean createArtikel(Artikel artikel)
  • Artikel findArtikel(int artikelNr)
  • List Artikel()

In dieser Aufgabe wird der Name Artikel verwendet, der dem Namen Artikel auf der Rechnung entspricht. Haben Sie in den Aufgaben 1 und 2 einen anderen Namen verwendet, so können Sie letzteren verwenden.

Ziel dieser Aufgabe ist es eine kleine Web-Anwendung zu schreiben, die folgende Funktionen erfüllen soll:

  • Artikel auflisten
  • Artikel anzeigen
  • Artikel erfassen

Nachfolgend werden die Webseiten dargestellt, die Sie für die Anwendung verwenden sollten. Die Seiten können Sie gemäss Ihren Vorstellungen gestalten.
